## Objective
Predict whether a loan applicant is likely to default using machine learning models.

## Steps Performed
1. Data Cleaning: handled missing values, encoded categorical data, scaled numeric features.
2. Exploratory Data Analysis: visualized key features like LoanAmount, Education, ApplicantIncome.
3. Model Training: Logistic Regression and Decision Tree.
4. Model Evaluation: accuracy, confusion matrix, and classification report.
5. Conclusion: Logistic Regression performed slightly better with higher accuracy.
